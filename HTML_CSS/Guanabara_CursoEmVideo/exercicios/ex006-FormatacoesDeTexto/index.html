Principais Formatações de Texto

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não há semântica).

Aqui temos um termo em destaque usando a tag <strong> (com semântica).

Itálico / Ênfase

Agora temos um termo em itálico usando a tag <i> (não há semântica).

E de forma similar, temos um termo em ênfaseusando a tag <em> (com semâmtica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Temos aqui um texto grande usando a tag <big>.

E um texto pequeno usando a tag <small>.

Texto deletado

Podemos marcar um texto como excluídopara indicar que ele deve ser lido, mas não considerado. Para isso usamos a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. É implementado com o uso da tag <ins>.

Também é possível sublinhar o texto com a tag <u>, mas não há semântica.

Texto sobrescrito

Útil para inserir expressões como: x20 + 3. O 20 está entre as tags <sup>20</sup>.

Texto Subscrito

É útil para inserir coisas do tipo: H2O. O número dois está entre as tags: <sub>2</sub>.
