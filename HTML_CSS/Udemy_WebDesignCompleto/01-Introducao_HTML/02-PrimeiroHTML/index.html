O head(h1, h2, h3, h4, h5, h6), mais do que tamanho, indica a hierarquia entre o conteúdo.
Sendo h1 o maior e h6 o menor nível de hierarquia.

Exemplo de lista não ordenada:

  • Item A
  • Item B
  • Item C

Exemplo de lista ordenada:

  1. Item A
  2. Item B
  3. Item C

Os links são definidos com a tag <a> <a/>.
Ela deve conter a url como valor do atributo href. E também pode ter o atributo target="_blank" para indicar que o link deve ser aberto em outra janela.

<a href="LINK_INTERNO">
Acesse um currículo vitae exemplo (link interno) CLICANDO AQUI

<a href="https://google.com" target="_blank">
Acesse a página inicial do Google (link externo) CLICANDO AQUI
